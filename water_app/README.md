# water_app

A Flutter-based mobile application that records audio and detects water-related activities using an AI backend.

## Getting Started

This app allows you to record sound from your phone and detect water activities like filling, cleaning, or idle.

Follow the steps below to run the project:

---

### 1. Enable Developer Mode on Phone

* Go to **Settings → About Phone**
* Tap **Build Number 7 times**
* Enable **Developer Options**
* Turn ON **USB Debugging**

---

### 2. Connect Phone

* Connect your phone using USB cable
* Allow all permissions on phone

---

### 3. Open Project in Terminal

cd water_app

---

### 4. Run Backend Server

python3.11 server.py

Keep this terminal running

---

### 5. Run Flutter App

Open another terminal:

flutter run


 Do NOT remove USB
 Wait about 1 minute
App will automatically open on your phone

---

### 6. Give Permission

* Allow **Microphone access**

---

## How to Use

1. Press **Start Recording**
2. Record water activity (10–30 seconds)
3. Press **Stop Recording**
4. Audio file saves automatically
5. App displays result:
Activity: filling
Confidence: 0.87


## Important Notes

* Keep server running
* Phone and laptop must be on same WiFi
* After app opens, USB can be removed
* Record clear sound for best results
